# WeddingWebsite

## Wedding Website Visual Draft — Option 2 (Full Site Shell)

This repository contains the first full-site visual/content draft for **Nia & Joshua's** wedding website.

- **Style:** Romantic Editorial Mediterranean Noir
- **Palette:** Pearlescent cream, burgundy, maroon, olive
- **Location:** Spain (final venue pending guest count)
- **Wedding dates:** **August 26–28, 2027**
- **RSVP deadline (final):** **June 10, 2027**

---

## Global Design System

### Color Palette
- Pearl Cream: `#F3EEE6`
- Shimmer Cream: `#EAE2D6`
- Burgundy: `#6A1E33`
- Deep Maroon: `#4A1423`
- Olive: `#5F674A`
- Warm Charcoal: `#2B2624`

### Typography
- Primary: elegant editorial serif
- Accent: romantic script (names, short headings, date lines)
- Body: readable serif

### UI Styling Notes
- Dark/moody base sections balanced with luminous cream cards
- Floral line-art corners and ornamental dividers
- Invitation-inspired bordered cards
- Gentle transitions (fade/slide)
